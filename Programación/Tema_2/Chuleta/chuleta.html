1. Arrays 1.1 Creación de Arrays [] Crea un array literal. Ejemplo: let numeros = [1, 2, 3] new Array() Crea un array vacío o con tamaño inicial. Ejemplo: let lista = new Array(5) 1.2 Acceso y Modificación array[index] Accede o modifica una posición del array. Ejemplo: frutas[1] = "pera" array.at() Accede a elementos usando índices negativos. Ejemplo: frutas.at(-1) 1.3 Propiedades array.length Devuelve o modifica la longitud del array. Ejemplo: console.log(frutas.length) 1.4 Añadir Elementos push() Añade elementos al final. Ejemplo: frutas.push("mango") unshift() Añade elementos al inicio. Ejemplo: frutas.unshift("fresa") 1.5 Eliminar Elementos pop() Elimina y devuelve el último elemento. Ejemplo: numeros.pop() shift() Elimina y devuelve el primer elemento. Ejemplo: frutas.shift() 1.6 Recorrer Arrays for Recorre un array con índice. Ejemplo: for (let i = 0; i < nums.length; i++) console.log(nums[i]) while Recorre un array mientras se cumpla una condición. Ejemplo: let i = 0 while (i < nums.length) i++ for...of Recorre los valores del array. Ejemplo: for (let fruta of frutas) console.log(fruta) 2. Ejemplo: sala[2][1] = "Reservado" 2.3 Recorrido Doble for Recorre filas y columnas de una matriz. Ejemplo: for (let fila of sala) for (let asiento of fila) console.log(asiento) 2.4 Funciones útiles con matrices Contar elementos Cuenta elementos disponibles. Ejemplo: sala[f][c] = "Reservado" 3. Strings 3.1 Creación "texto" String con comillas dobles. Ejemplo: let msg = "Hola" 'texto' String con comillas simples. Ejemplo: let saludo = 'Bienvenido' texto Template literal. Ejemplo: `Hola ${nombre}` 3.2 Concatenación + Une texto manualmente. Ejemplo: "Hola " + nombre Template literals Concatena variables fácilmente. Ejemplo: `Hola ${nombre}` 3.3 Modificación toUpperCase() Convierte texto a mayúsculas. Ejemplo: nombre.toUpperCase() toLowerCase() Convierte texto a minúsculas. Ejemplo: nombre.toLowerCase() slice() Extrae parte de una string. Ejemplo: texto.slice(2, 5) replace() Reemplaza la primera coincidencia. Ejemplo: frase.replace("hola", "adiós") replaceAll() Reemplaza todas las coincidencias. Ejemplo: texto.replaceAll("a", "x") 3.4 División y Búsqueda split() Divide una string en un array. Ejemplo: lista.split(", ") indexOf() Devuelve la posición de una palabra. Ejemplo: texto.indexOf("JavaScript") includes() Comprueba si contiene un texto. Ejemplo: texto.includes("hola") startsWith() Comprueba inicio de texto. Ejemplo: texto.startsWith("Hola") endsWith() Comprueba fin de texto. Ejemplo: texto.endsWith("mundo") 3.5 Limpieza y Conteo trim() Elimina espacios del inicio y final. Ejemplo: entrada.trim() string.length Devuelve número de caracteres. Ejemplo: texto.length 3.6 Ejemplo práctico Construcción de mensaje Procesa y muestra un saludo. Ejemplo: `¡BIENVENIDO, ${nombre}!`
